# AquaWare - Next-Gen Water Quality Management Platform

**Control of Indirect Industrial Discharges**

**Live Demo:** [https://shadmansakibrahman.github.io/Aquaware-Website-V4/](https://shadmansakibrahman.github.io/Aquaware-Website-V4/)

AquaWare is a data-driven platform that provides governments with immediate insight into potential sources of SVHCs (Substances of Very High Concern) and WFD (Water Framework Directive) priority substances, enabling proactive action towards WFD objectives.

> *"With AquaWare we want to build a future together in which clean water is the standard."*

---

## About

AquaWare develops digital solutions that translate complex water data into actionable insights. With smart analyses, interactive dashboards, and clear visualizations, we make information directly applicable in practice — enabling governments to act data-driven and with insight.

**AquaWare** stands for:
- **Aqua** — for water, our most valuable natural system
- **Ware** — refers to software and digital tools
- **Aware** — creating awareness about water quality and the impact of industrial discharges

---

## Features

### Centralized Database
A single, central database for every business in your area — including Chamber of Commerce profile, location, discharge routes, expected substances, risk scores, and administrative information.

### Interactive Dashboards
Filter in seconds by company data, discharge points, sectors, substances, and WWTPs. View key figures and trends through clear charts, tables, and GIS maps. Everything is searchable and exportable.

### Priority Lists
All companies matching selected filters appear in a clear overview with core details and expected substances. Sort and prioritize by risk, and export to CSV/Excel for further action.

### Company Reports
One complete dossier per company with all available information in one place. Exportable to PDF and CSV for internal alignment and decision-making.

### Heatmaps
Visualize the spread and intensity of sectors, substance groups, and specific substances. Hotspots become immediately visible per area, supporting risk-based and targeted supervision.

### Knowledge Hub
A comprehensive knowledge base featuring blogs and insights on water quality topics including SVHCs, PFAS, indirect discharges, data-driven water management, and discharge permit regulations.

### Newsletter Subscription
Stay informed with product updates and water-quality insights delivered to your inbox.

### Contact Form
Built-in contact form with direct email and phone support for inquiries.

## Government Support

AquaWare supports key government activities:

- **Regulation and Compliance** — Data-driven insights to get businesses on the radar faster and more targeted
- **Policy Development** — Combining trends, risk zones, and sector analyses for substantiated and future-proof water quality policy
- **Risk Assessment** — Identify business risks and analyze which companies are likely to discharge ZZS/KRW substances
- **Source Detection** — Trace discharge routes back from WWTP hotspots and match them with sectors and substance groups

---

## How It Works

AquaWare uses a funnel-based approach:

1. **All Companies in View** — Identify all companies in your area with linked data
2. **Water-Relevant Companies** — Filter to those truly relevant to water quality based on business profile and discharge risks
3. **Prioritization** — Generate a data-driven prioritization list based on risk profile

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/ShadmanSakibRahman/Aquaware-Website-V4.git

# Navigate to project directory
cd Aquaware-Website-V4

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
npm run preview
```
